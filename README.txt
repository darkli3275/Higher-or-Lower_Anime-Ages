README

    Higher or Lower? Anime Ages! is a game in which you guess whether a character is older or younger than another
    character.

    Open Launcher.jar to open the game. Ensure Launcher.jar and data are in the same folder. Essential files in
    data are listed in FILE MANIFEST.


FILE MANIFEST

    data/game -> CharacterImages, CharacterData.json
    data/local -> checkmark.png, endPageBG.png, Scores.json, startPageBG.png, x_mark.png
